### Prerequisites

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/codax69/sever-main.git
   cd sever-main
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the server:**

   ```bash
   node index.js
   ```

